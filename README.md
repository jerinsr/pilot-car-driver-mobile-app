# Pilot Car Driver Mobile App

A React + Vite mobile-first web app for pilot car drivers. The original design is available at [Figma](https://www.figma.com/design/Safz09kpmxMncDwQ3LFgTM/Pilot-Car-Driver-Mobile-App--Copy-).

## Tech stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS 4
- Radix UI / shadcn-style components
- MUI, Recharts, motion, react-router

## Getting started

```bash
npm install
npm run dev
```

The dev server runs at **http://localhost:5599** (configured in `vite.config.ts`).

## Scripts

| Command           | Description                              |
| ----------------- | ---------------------------------------- |
| `npm run dev`     | Start dev server on port 5599            |
| `npm run build`   | Production build to `dist/`              |
| `npm run preview` | Preview the production build (port 5599) |
